BIH-HIS Design System

Design System Proposal

Proposed standard for colors, spacing, and typography across the BIH-HIS application.

1. Color roles

Every color has a purpose. The same role is reused everywhere it applies buttons, banners, tables, forms so the product feels consistent.

Swatch Hex Role Purpose
#FFFFFF Background Main screen surface
#FFFFFF Card Surface of every panel, form, table
#252525 Foreground Main reading text
#787878 Muted Helper text, captions, placeholders
#EBEBEB Border Frame around cards and inputs
#B5B5B5 Focus ring Highlight around the focused input

Status colors

Swatch Solid hex Solid Swatch Soft hex Soft tint Used for
#3BA85C Success #DFEEDE Success soft Confirmations, completed states
#D49520 Warning #F5E9D4 Warning soft Caution, expiring items, "Are you sure?"
#E2422C Danger #F5DCD6 Danger soft Errors, delete confirmations
#4280DC Info #E0EAF5 Info soft Tips, neutral information panels

4. Typography

Five text roles. Every piece of text in the app uses one of these never a custom size.

Role Size Weight Line height Used for
Title 20 px 600 (semibold) 1.2 Page titles
Heading 18 px 600 (semibold) 1.3 Card and section headings
Label 14 px 500 (medium) 1.4 Form labels, table column headers
Body 14 px 400 (regular) 1.5 Default body text, table cells
Caption 12 px 400 (regular) 1.4 Helper text, footnotes, badge text
